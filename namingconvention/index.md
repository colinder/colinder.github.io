# Naming Convention

>정해진 규정은 없으나, 많은 개발자들이 암묵적으로 지키고 있는 이름짓는 방법.

​	

## Python

- snake_case 🐍

  ```python 
  # _ == snake
  some_var = 5
  my_awesome = 3
  ```
  
  

## Javascript, Vue

- camelCase 🐪

  ```bash
  # 낙타등이 중간에 솟아 있는 모습
  someVar = 5
  pyAwesome = 3
  ```
  
  
  
- uppercamelCase 🐪🐪

  ```bash
  # 낙타얼굴과 등이 중간에 솟아 있는 모습
  SomeVar = 5
  PyAwesome = 3
  ```
