# Naming Convention

>정해진 규정은 없으나, 많은 개발자들이 암묵적으로 지키고 있는 이름짓는 방법.

​			

- snake_case 🐍

  ```python 
  # _ == snake
  some_var = 5
  my_awesome = 3
  ```
  
  ​	
  
- camelCase 🐪

  ```bash
  # 낙타등이 솟아 있는 모습
  someVar = 5
  pyAwesome = 3
  ```

  ​	

- uppercamelCase 🐪🐪

  ```bash
  # 낙타얼굴과 등이 중간에 솟아 있는 모습 (파스칼식(Pascal case)이라고도 불린다.)
  SomeVar = 5
  PyAwesome = 3
  ```

  ​	

- Kebab-Case 🥙

  ```bash
  # 케밥이 꼬챙이에 꽂힌 모습에서 생긴 방법(이라는데 난 생김새는 잘 모르겠다.ㅎ)
  main-login = 5
  feed-content = 1
  ```
