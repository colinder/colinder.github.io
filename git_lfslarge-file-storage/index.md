# Git LFS(Large File Storage) 사용법

> Git은 **개당 파일에 100MB**의 용량 제한이 걸려 있다. (**전체 용량 제한은 없다.**) 다만, 프로젝트를 하다보면 대용량의 자료나 파일이 생기는 경우가 있다. 이때 용량 제한의 문제를 해소할 수 있는 방법이 LFS이다.
>
> LFS는 별도로 설치하여 사용하면 되며, 방법을 정리한다.
